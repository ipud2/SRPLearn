# SRPLearn

基于Unity可编程渲染管线造轮子。 主要目的是为了边写边学习，不考虑平台兼容和性能优化等问题。


Unity版本: 2019.4.16

注意，项目不需要用到`com.unity.render-pipelines.universal`，可以将其从包依赖中删除。
